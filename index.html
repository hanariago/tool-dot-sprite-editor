도트 스프라이트 에디터

브라우저에서 바로 쓰는 무료 픽셀 아트 / 스프라이트 에디터. 설치도 회원가입도 없이, 모든 작업은 당신의 브라우저 안에서만 처리됩니다.

이렇게 쓰세요

위 캔버스에 바로 도트를 찍어보세요. 브러시·대칭·선/도형으로 그리고, 프레임을 추가해 애니메이션(걷기·공격 등)을 만들고, 어니언 스킨으로 직전 동작을 비춰보며 다듬습니다. 완성하면 PNG(배율 선택)나 스프라이트 시트로 내보내거나, .sprite 파일로 저장해 나중에 이어서 편집하세요. 사진·이미지를 끌어다 놓으면 배경 제거·팔레트 감소를 거쳐 도트로 변환됩니다.

개인정보

그리기·저장·내보내기 등 모든 핵심 기능은 서버 없이 브라우저에서만 동작하며 아무것도 업로드하지 않습니다. AI 그리기 기능은 선택사항으로, 사용 시 당신이 입력한 API 키로 당신이 고른 AI 서비스에 직접 전송되며 이 사이트의 서버는 거치지 않고 아무것도 저장하지 않습니다.

자주 묻는 질문

무료인가요?

네, 완전 무료이며 회원가입이 필요 없습니다. 오픈소스입니다.

내 그림이 서버로 전송되나요?

아니요. 그리기·저장·내보내기는 모두 브라우저 안에서만 처리됩니다. AI 기능을 켤 때만, 당신의 키로 당신이 고른 AI에 직접 전송됩니다.

어떤 형식으로 저장/내보내나요?

편집을 이어가려면 .sprite(JSON, 프레임 포함), 결과물은 투명 PNG(배율 ×1~16) 또는 스프라이트 시트 PNG로 내보냅니다.

모바일에서도 되나요?

네. 터치로 그리기·핀치 줌을 지원하며 화면 폭에 맞춰 레이아웃이 조정됩니다.

애니메이션도 만들 수 있나요?

네. 여러 프레임을 만들고 FPS로 재생하며, 어니언 스킨과 모션 템플릿으로 걷기·공격 같은 동작을 그릴 수 있습니다.

제 API 키는 안전한가요?

AI 기능은 선택이며, 키를 넣지 않아도 모든 그리기·저장·내보내기를 쓸 수 있습니다. 입력한 키는 이 브라우저에만 저장되고(이 사이트 서버로 전송·저장되지 않음), 요청은 당신이 고른 AI 제공사로만 직접 전송됩니다. 코드는 오픈소스라 직접 확인할 수 있으니, 신뢰가 서는 경우에만 본인 책임으로 키를 입력하세요. 공용 PC에서는 사용 후 설정에서 키를 비우는 것을 권장합니다.

🤖 AI 도트 그리기

⚠️ 키는 이 브라우저에만 저장되고 당신이 고른 AI로만 직접 전송됩니다(이 사이트 서버 경유·저장 없음). 오픈소스라 검증 가능 — 신뢰가 설 때만 본인 책임으로 입력하세요. 키 없이도 위 「프롬프트 복사」 방식으로 쓸 수 있어요.
다른 PC면 그 PC의 주소 입력 (예: http://192.168.0.10:8642/v1)
🔒 키·서버 없이 — 내 구독(ChatGPT/Claude/Grok·Nano Banana) 채팅에 붙여넣어 받아옵니다.
② 생성된 이미지를 저장한 뒤 → 캔버스에 드래그&드롭(또는 아래 버튼)하면 '이미지 임포트'에서 도트로 변환됩니다.
