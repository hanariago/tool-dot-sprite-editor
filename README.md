# 🎨 Dot Sprite Editor (도트 스프라이트 에디터)

> 무료 · 브라우저 단독 · 픽셀 아트/스프라이트 에디터 — Free, browser-only pixel art & sprite editor.

**🔗 Live:** https://hanariago.github.io/tool-dot-sprite-editor/

설치도 회원가입도 없이 브라우저에서 바로 도트를 찍고, 프레임 애니메이션을 만들고, PNG·스프라이트 시트로 내보냅니다. 모든 핵심 기능은 서버 없이 브라우저에서만 동작합니다(업로드 없음).

## ✨ 기능 / Features
- **그리기**: 브러시(1~3px) · 대칭(좌우/상하/사방) · 선/사각형/원 · 채우기 · 스포이드
- **편집**: undo/redo(크기·배율 포함) · 선택영역 이동/복사 · 반전 · 90° 회전 · ×N/÷N 배율
- **애니메이션**: 프레임 추가/복제/삭제 · 어니언 스킨 · FPS 재생 · 모션 템플릿 · 스프라이트 시트
- **색/팔레트**: HEX·HSL·불투명도 · 8~256색(웹세이프216+회색40)·무제한 · 최근색 · 팔레트 저장
- **이미지 임포트**: 사진/PNG·AI 생성 이미지 → 도트 변환 (안티앨리어싱·가장자리 선명도·팔레트 감소·배경 제거·**도트 블록 N×N 묶기**로 유사색 정리)
- **AI 그리기(선택)**: 본인 키로 Claude/GPT/Grok/Gemini/Hermes, 또는 프롬프트 복사+붙여넣기, Nano Banana 이미지 생성
- **저장**: `.sprite`(JSON, 다프레임) · PNG(배율 ×1~16) · 스프라이트 시트 PNG · 임시저장(로컬)
- **i18n**: 한국어 / English (확장형), 모바일 대응

## 🔒 개인정보 / Privacy
그리기·저장·내보내기 등 모든 핵심 기능은 브라우저에서만 동작하며 아무것도 업로드하지 않습니다. AI 기능 사용 시에만, **당신이 입력한 키로 당신이 고른 AI 서비스에 직접** 전송되며 이 사이트 서버는 거치지 않습니다.

> **AI 키 고지:** AI 기능은 선택입니다(키 없이도 모든 그리기·저장·내보내기 가능). 입력한 키는 이 브라우저(localStorage)에만 저장되고 이 사이트 서버로는 전송·저장되지 않으며, 요청은 당신이 고른 AI 제공사로만 직접 갑니다. 코드는 오픈소스라 직접 검증할 수 있으니 **신뢰가 설 때만 본인 책임으로** 키를 입력하세요. 공용 PC에서는 사용 후 설정에서 키를 비우길 권장합니다.

## 🚀 실행 / Run
- 온라인: 위 Live 링크
- 로컬: `index.html`을 더블클릭(`file://`) 또는 `python -m http.server 8000`

## 🛠 기술 / Tech
바닐라 JavaScript · HTML5 Canvas · CSS · 빌드 도구 없음 · 일반 `<script>` 순차 로드.

## 사용한 오픈소스 / 에셋
- 외부 라이브러리·폰트·번들 에셋 없음 (순수 HTML/CSS/JS). AI 그리기는 사용자가 선택한 외부 AI 서비스(Anthropic/OpenAI/xAI/Google/Hermes)를 본인 키로 직접 호출하는 것으로, 코드에 포함된 모델·키는 없습니다.
